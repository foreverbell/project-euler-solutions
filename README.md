## Build

We provide two ways to build one solution.

### makefile
```
$ make 42
$ Build/42
```
If ghc is complaining about missing packages, see `project-euler-solutions.cabal.in` and `configure` for more details.

### stack
```
./configure 42
stack init
stack build
stack exec 42
```
